Experimentation Matrix 2

This matrix will be for making Hydromel or Standard strength Meads. The purpose is to learn to control fermentation to make Meads at a given strength.

    Experimentation Matrix 1

    July 18, 2025

    This matrix contains my earliest batches where I'm learning by making mistakes and then making corrections. The main goal is to make some Mead from each local Honey I can easily obtain and see what they taste like. I'm making sack strength Mead because it sounds fun to make Mead with a lot of alcohol.

    I realized somewhat after the fact that what I'm doing here is "capping out" the yeast. This is a reasonable enough strategy but based on what I'm seeing using Lalvin ICV D47 Yeast I could cap at 18+% which is perhaps too much.
